Artifacts:

The artifacts in this portfolio showcase my work in software design and engineering, algorithms and data structures, and databases. Throughout the course, I have refined these artifacts by enhancing their functionality, increasing their complexity, and incorporating comments to improve code readability and maintainability.

Each enhancement in this portfolio highlights a project originally developed for CS-340, completed on June 23, 2024, which focused on building an interactive database for rescue animal centers. Written in Python, the project utilizes PyMongo to interact with MongoDB, serving as the backend for performing essential CRUD (Create, Read, Update, Delete) operations on animal shelter data. This artifact demonstrates my skills in Software Engineering and Design, Algorithms and Data Structures, and Database Management, showcasing how I refined and expanded its functionality across these core areas to create a robust and maintainable solution.

This artifact was chosen because it exemplifies the transformation of a Python-based project with MongoDB and Dash into a more advanced full-stack web application, utilizing Spring Boot for the backend and React for the frontend. This evolution highlights my proficiency in software development principles and modern technologies, demonstrating my ability to refactor and enhance the project into a scalable, maintainable, and robust solution.
